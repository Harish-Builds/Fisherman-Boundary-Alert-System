# 🚨 Fisherman Boundary Alert System

## 📌 Overview
A GPS-based safety system designed to alert fishermen when approaching restricted maritime boundaries.

---

## 🎯 Problem Statement
Fishermen unknowingly crossing borders face legal issues and safety risks.

---

## 💡 Solution
A GPS-enabled system that detects boundaries and alerts users using a buzzer.

---

## ⚙️ Features
- Real-time GPS Tracking
- Geofencing (Boundary Detection)
- Buzzer Alert System
- Safety Notifications

---

## 🔌 Pin Configuration

| Component     | Pin |
|--------------|----|
| GPS TX       | D1 |
| GPS RX       | D2 |
| Buzzer       | D5 |

---

## 🚀 How to Run
1. Upload GPS code
2. Connect GPS module
3. Define boundary coordinates
4. Monitor alerts

---

## 📈 Applications
- Marine Safety
- Fishermen Protection
- Coastal Monitoring

---

## 🛠 Tech Stack
- Arduino / ESP8266
- GPS Module (NEO-6M)
- Buzzer
